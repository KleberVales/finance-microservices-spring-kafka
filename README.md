# Finance Microservices - Spring Kafka


### Start the system

docker compose up --build

This will start:

- Kafka
- Zookeeper
- PostgreSQL databases
- All microservices

---

| Service | Port |
|------|------|
| auth-service | 8082 |
| user-service | 8081 |
